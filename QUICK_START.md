## üîß CUSTOMIZATION

### Change Products
1. Open `script.js`
2. Find `const products = [...]`
3. Add/edit products with:
   - name, price, category
   - rating, reviews, discount
   - description, icon
   - images, highlights

### Change Colors
1. Open `style.css`
2. Find `:root { ... }` (lines 1-40)
3. Update CSS variables:
   - `--primary`: Main color
   - `--secondary`: Accent color
   - `--success`: Success color

### Change AI API Key
1. Open `script.js`
2. Find `const API_KEY`
3. Replace with your Google Gemini API key
