# KAI Website - Quick Start Guide

## Get Started in 30 Seconds

### Step 1: Open the Website
Simply double-click: index.html

### Step 2: Start Chatting
1. Click "Start Chat" button on homepage
2. Type any product query (e.g., "smartphone under Rs.30,000")
3. KAI will recommend products and help you shop!

## Your Files

- index.html  - Main website (open this!)
- style.css   - Beautiful vibrant design
- script.js   - AI chatbot functionality

## What Can KAI Do?

### Smart Chat
- Ask about ANY product
- Get instant recommendations
- Compare products
- See prices in INR

### Product Discovery
- Search smartphones, laptops, headphones, etc.
- Get 2-3 similar options
- See which is the BEST to buy

### Mock Orders
- Say "I want to buy [product]"
- Get instant order confirmation
- Track in Orders section

## Technical Info

AI Powered By: Google Gemini API
API Key: Already configured
Storage: Browser localStorage
Internet: Required for AI responses
Fallback: Works offline with demo data

## Launch Options

Option 1: Direct (Easiest)
Double-click index.html

Option 2: Local Server
python3 -m http.server 8000
Then open: http://localhost:8000

Option 3: Start Script
./START_KAI.sh
