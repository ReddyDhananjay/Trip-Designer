## Animation Effects

### 1. Floating Background Shapes
- 4 large circular gradient orbs
- Slowly float up and down
- Rotate slightly
- Heavy blur effect (60px)
- Colors: Red, Cyan, Purple, Yellow
- 20-second animation cycle

### 2. AI Avatar Animation
- Robot emoji (ğŸ¤–) in center
- Gradient background (Purple to Pink)
- Gentle floating motion (3s cycle)
- Two pulsing rings around it
- Rings fade in/out and scale
- Creates "breathing" effect

### 3. Card Hover Effects
```
Normal State:
- Flat on page
- Subtle shadow
- Semi-transparent background

Hover State:
- Lifts up (translateY: -10px)
- Shadow expands
- Background lightens
- Smooth 0.3s transition
```

### 5. Button Animations

- Gradient backgrounds pulse
- Icons bounce

## Responsive Breakpoints

### Mobile (< 768px)
- Single column layout
- Stacked navigation
- Full-width cards
- Smaller hero text (2.5rem)
- Reduced avatar size

### Tablet (768px - 1024px)
- Two column grid
- Adjusted spacing
- Medium-sized elements

### Desktop (1024px+)
- Full grid layouts
- Side-by-side hero content
- Optimal spacing
- All animations enabled

### Large (1440px+)
- Max-width containers (1400px)
- Extra spacing
- Large displays optimized

```

```

### Gradient Text

## User Interaction Feedback

### Hover States
- Cursor changes to pointer
- Element lifts or scales
- Colors brighten
- Shadows expand

### Active States
- Slight scale down
- Quick transition
- Visual click feedback

### Focus States
- Outline or border highlight
- Increased contrast
- Keyboard navigation friendly

### Loading States
- Spinner animation
- Typing indicator dots
- Skeleton screens

## Accessibility Features

- High contrast ratios
- Focus visible states
- Semantic HTML structure
- ARIA labels where needed
- Keyboard navigation support
- Screen reader friendly
